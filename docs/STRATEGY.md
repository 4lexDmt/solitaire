# Aevanor Solitaire — Product Strategy (locked)

**Status:** locked Jul 2026  
**Site:** [aevanor.com](https://aevanor.com)  
**UI:** Windows 9x desktop shell (`DesktopShell`) — do not replace with baize/mobile-home chrome.

## One-liner

> Be Solitr’s craft and focus, with Solitaired’s daily/stats habit, plus fairness/offline — in an original Aevanor Win9x desktop — and never become an ad-funded encyclopedia.

## Positioning

| Axis | Choice |
|---|---|
| Catalog | **Curated solitaire families** — Solitaire, FreeCell, Spider, Pyramid, TriPeaks, Yukon, Golf. Adjacent Patience next; not Hearts/Yahtzee sprawl. |
| Feel | **Win9x desktop chrome** + original card art. Polish and nostalgia beat page count. |
| Fairness | **Real solver + optional winnable-only deals** for Klondike. Honest messaging, no “rigged” feel. |
| Ads | **Ad-free by default** — brand wedge vs CardGames.io / Solitr / Solitaired. |
| Offline | **PWA + IndexedDB first**; cloud sync optional. |
| Retention | **Daily challenge, streaks, local stats** — Solitaired loops without coins/spam. |
| SEO | **Per-variant indexable routes** with rules/strategy copy under the play surface. |

## Explicit non-goals (for now)

- 500+ obscure Patience titles for SEO volume
- Trick-taking / multiplayer suite
- In-game ads or “watch ad for hint”
- Replacing Win9x chrome with a generic mobile home shell
- Cross-selling the Aevanor caregiver / Aida product inside the game

## Competitive stance

- **vs Solitr:** match focus; win on craft, offline, fairness, daily, Win9x identity.
- **vs CardGames.io:** do not chase suite breadth; keep anti-bloat discipline.
- **vs Solitaired:** borrow SEO landings + daily prominence; skip ad wall and encyclopedia catalog.

## Expansion order

1. SEO + distribution (share / install / bookmark) — this pass
2. Fairness / winnable messaging in product copy — this pass
3. Paid extras only after traffic (see [MONETIZATION.md](./MONETIZATION.md))
4. More solitaire families before any non-solitaire game
5. Native wrap only after web retention is proven
