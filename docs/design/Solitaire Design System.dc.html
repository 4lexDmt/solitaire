Accessibility

WCAG contrast ratios computed per theme (target ≥ 4.5:1). Suits are distinguishable by shape; the four-color deck (above) adds hue. Focus rings use --accent; touch targets ≥ 44×44.

{{ contrastReport }}
One flag. --ink-red on --card-face in Midnight computes to 3.57:1 — below 4.5. Red ink is only ever drawn as large bold glyphs (rank cap-height ≈46, pips ≈30+), where the WCAG large-text threshold is 3:1, so it passes for its actual usage. If you ever set red body text, nudge --ink-red darker in Midnight — but per the contract I haven't changed the value. The four-color deck's blue/green also clear shape-independent distinction.
